# Customer Churn Analysis

## Project Overview
This project analyzes customer churn behavior and retention trends to identify factors affecting customer loss.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insights
- Month-to-month contracts showed higher churn
- Long-term customers had better retention
- Short-tenure customers were more likely to churn
- Customer retention strategies can reduce business loss
